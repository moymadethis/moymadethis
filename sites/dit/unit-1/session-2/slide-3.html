Skill level

A user interface developer also needs to consider the ability of the target audience to access technology. In Session 1, you learned about the digital divide and factors that can create a void between those that can and those that can’t use technology…skill level is one of these factors. Basically, users will have a range of skills and a level of ability which will determine how effectively they are able to interact with an interface.

Click on the images below to learn about these different skill levels and the impact they can have on interface design.

Novice user

  • Little to no experience using digital devices or interfaces
  • Might need supervision and/or assistance using digital technology
  • Might not understand why they are using the interface or know how to complete simple tasks

Occasional user

  • Some experience of using digital devices in a semi-regular fashion
  • They can complete basic everyday tasks using common digital interfaces
  • Some tasks might need to be completed multiple times before processes are learnt
  • Can use and navigate an interface but might not be fluid or quick in doing so

Regular user

  • Experience in completing everyday tasks on different digital devices using different user interfaces
  • A regular user can independently learn a new device or interface through trial and error or using help guidance
  • Help will be needed occasionally with more complex systems and interfaces

Expert user

  • Vast experience in using a wide range of devices and interfaces on a regular basis
  • Use and access will be second nature and instinctive
  • Will be able to know what a tool does before using it
